# Keyboard shortcuts

## Runtime

| Key | Action |
| --- | --- |
| **F5** | Play (run the current graph) |
| **F6** | Stop |
| **Esc** (anywhere, any app) | Global kill-switch — stops a running script from any focused window |

## Editing the graph

| Key | Action |
| --- | --- |
| **Delete / Backspace** | Remove selected nodes |
| **Ctrl+C** | Copy selected nodes |
| **Ctrl+X** | Cut selected nodes |
| **Ctrl+V** | Paste copied nodes |
| **Ctrl+D** | Duplicate selected nodes |
| **Ctrl+Z** | Undo |
| **Ctrl+Y** / **Ctrl+Shift+Z** | Redo |

## Visualizer

| Action | Effect |
| --- | --- |
| **Single click** on a pixel | Copy colour as `0xRRGGBB` to clipboard; if a compatible node is selected, auto-set its `target` param |
| **Click + drag** | Rubber-band selection (ROI) |
| **Esc** (with visualizer focus) | Clear current rubber-band |
| **Mouse wheel** | Zoom in / out |
| **Ctrl+0** | Reset zoom to fit |

## File

| Key | Action |
| --- | --- |
| **Ctrl+N** | New script (seeded with On Start) |
| **Ctrl+O** | Open existing `.rvscript` |
| **Ctrl+S** | Save |
| **Ctrl+Shift+S** | Save as… |
| **Ctrl+Q** | Quit |
